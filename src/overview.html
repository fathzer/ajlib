This is a simple java library with Swing widgets and other stuff.

This library tries to simplify the development of Swing applications ... which is sometime a nightmare !
It contains the classes I developed during the Yapbam project in order to deal with the problems I encountered.

This work is still under progress and some things may be changed in the near future (especially some class names).
The classes available here are quite stable and used in the Yapbam project.
